# Everdrop Pure Mineral Water

Premium mineral water delivery website for **Everdrop Pure Mineral Water** — Islamabad & Rawalpindi, Pakistan.

Built with **Next.js** (App Router), **TypeScript**, **Tailwind CSS**, **shadcn/ui**, and **GSAP** scroll animations. Configured for **static export** — deploy the `/out` folder to any CSR/static host.

## Quick Start

```bash
npm install
cp .env.example .env.local   # set NEXT_PUBLIC_SITE_URL
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Production Build & Deploy

```bash
npm run build
```

Upload the contents of the **`/out`** folder to your hosting web root (e.g. `public_html`).

| Setting | Value |
|---------|-------|
| Build output | `/out` |
| Static export | Yes — no Node.js server required |
| Trailing slashes | Enabled (`/about/` not `/about`) |

## Editing Content (No Code Required)

All business content lives in the **`/data`** folder:

| File | What to edit |
|------|--------------|
| `data/site.json` | Brand name, taglines, phone, email, address, social links, FAQ, values |
| `data/products.json` | Product names, prices (PKR), descriptions, images, WhatsApp messages |
| `data/delivery-areas.json` | Islamabad & Rawalpindi delivery sectors |
| `data/testimonials.json` | Customer reviews |
| `data/gallery.json` | Gallery images and captions |
| `data/navigation.json` | Header and footer menu links |

## Design System

UI rules are generated by the **ui-ux-pro-max** skill:

- Global rules: `design-system/everdrop-pure-mineral-water/MASTER.md`
- Page overrides: `design-system/everdrop-pure-mineral-water/pages/`

Regenerate after design changes:

```bash
python3 .cursor/skills/ui-ux-pro-max/scripts/search.py \
  "premium mineral water delivery" --design-system --persist \
  -p "Everdrop Pure Mineral Water" -f markdown
```

## Project Structure

```
data/              ← business content (JSON)
public/            ← static assets, OG image
src/app/           ← pages (static export)
src/components/    ← UI, GSAP animations, forms
src/lib/           ← content loader, SEO schema, WhatsApp helpers
design-system/     ← ui-ux-pro-max design rules
out/               ← deploy this folder
```
